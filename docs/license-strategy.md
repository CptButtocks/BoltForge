# License Strategy

The project license is not chosen yet.

Do not copy, link, or vendor GPL code until maintainers explicitly decide
whether BoltForge will use a GPL-compatible license. This matters most for
Wrench, which is useful public research material but GPL-3.0 licensed.

Before accepting broad outside contributions or incorporating third-party code,
choose and commit a repository `LICENSE` file, then update `THIRD_PARTY.md` and
`AGENTS.md` with the decision.

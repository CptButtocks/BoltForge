# Asset Format Notes

This directory is for project-authored notes about original data formats and
decoded runtime representations.

Expected topics:

- ISO and raw-sector loading,
- table-of-contents layouts,
- WAD headers and compression,
- textures and palettes,
- terrain, moby, tie, shrub, and collision geometry,
- gameplay instances and pvars,
- sound banks and VAG/989snd research.

Use confidence labels when a field is inferred rather than confirmed.
